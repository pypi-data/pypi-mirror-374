KEYBOARD SHORTCUTS

RECORDING
  Space        Toggle recording
  P            Play current recording
  O            Toggle monitoring

NAVIGATION
  Up/Down      Navigate utterances
  Left/Right   Browse takes

DISPLAY
  M            Toggle mel spectrogram & level meter
  I            Toggle info overlay
  F10          Toggle fullscreen

SECOND WINDOW (Shift + Key)
  Shift+M      Toggle meters in second window
  Shift+I      Toggle info panel in second window
  Shift+F10    Toggle fullscreen in second window

FILE OPERATIONS
  Ctrl+D       Delete current recording

DIALOGS
  Ctrl+F       Find utterance
  Ctrl+U       Utterance order
  Ctrl+I       Session settings

SESSION MANAGEMENT
  Ctrl+N       New session
  Ctrl+O       Open session
  Ctrl+Q       Quit

HELP
  F1           Show keyboard shortcuts
